The Problem

Training runs produce data nobody reads.

You log loss curves, gradient norms, attention entropy, weight norms. Hundreds of epochs of signal. You check if loss went down and move on.

LITMUS reads those artifacts autonomously. It finds phase transitions you missed, gradient events that predicted capability jumps, attention head specialization patterns, and grokking dynamics. Then it tries to debunk its own findings. Only what survives gets reported.

What LITMUS Finds

Grokking phase transition at epoch 47 Grade A / Surprise 0.91

"Train loss converges by epoch 20 but validation loss plateaus for 25 epochs, then drops 62% between epoch 45-50. Changepoint detection confirms structural break at epoch 47. Gradient norm spike at epoch 47 precedes the transition. Attention entropy drops 40% in the same window, consistent with head specialization driving generalization."

p-value: < 0.001 Cohen's d: 1.84 Checks passed: 5/5 Method: Changepoint + t-test
Gradient spikes as phase transition precursors Grade B / Surprise 0.67

"Gradient norm spikes at epoch 12 and 47. The epoch 47 spike precedes the grokking transition by 0-2 epochs. Cross-correlation peaks at lag -2 (r = 0.73). However, the epoch 12 spike has no corresponding phase transition, weakening the causal claim."

p-value: 0.008 Cohen's d: 0.64 Checks passed: 4/5 Failed: Temporal stability
Weight norm acceleration predicts loss independently Grade C / ARCHIVED

"Weight norm acceleration correlates with loss decrease (r = -0.41), but partial correlation controlling for gradient norm drops to r = -0.08. The signal is entirely explained by gradient dynamics. Archived: confounding variable explains the effect."

p-value: 0.12 Cohen's d: 0.22 Checks passed: 2/5 Failed: Confounder, Effect Size, FDR

Ranking

Surprise Score

Obvious confirmations get deprioritized even at p < 0.001. Only findings that are statistically robust AND genuinely unexpected surface to the top.

discovery_score = surprise * significance * effect_size
surprise = KL divergence from expected  |  significance = -log10(p_adj)  |  effect = |Cohen's d|
Track Fit

Operator Rubric Alignment

Agentic Autonomy

Zero human input after upload. Agent decides what to test, when to go deeper, when to prune. Five stages run without intervention.

Tool Use & Integration

Local numerical analysis (changepoints, z-scores, t-tests, correlation, entropy) feeds into Claude for interpretation. BH-FDR and effect size run locally. Hybrid computation.

Multimodal & Unstructured

Accepts CSV, JSON, YAML training artifacts. Handles different logging formats. Profiles data structure before analysis.

State & Context Mgmt

Profile summary carries through all stages. Experiment results feed validation. Findings accumulate and rank against each other. Full pipeline context preserved via SSE.
